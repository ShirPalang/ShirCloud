## Package's:
 - JSON Server
 - Wafesurfer.js

## Setup project:

### 1. install packages

-   ```
    cd database

    npm i
    ```

### 2. run JSON Server

-   ```
    json-server --watch db.json
    ```

### 3. and run project with Live Server
